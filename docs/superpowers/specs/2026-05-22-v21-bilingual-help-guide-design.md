# v2.1 双语帮助说明设计

## 目标

在 SettingsScreen 增加一个简单易懂的说明入口，帮助老师快速理解如何配置、导入题目、开始课堂游戏。说明同时提供英文和中文，可在弹窗内一键切换。

## 入口

- 位置：设置页右上角，标题牌右侧。
- 文案：`Need Help? ->`
- 按钮：一个圆形 `i` 图标按钮。
- 行为：点击后打开帮助弹窗；弹窗打开时拦截设置页原有点击，避免误触滑杆、导入按钮或开始按钮。

## 弹窗

- 风格：沿用羊皮纸卡片、木牌标题、像素按钮。
- 标题：
  - 英文：`Game Guide`
  - 中文：`游戏说明`
- 右上角按钮：
  - `中文` / `EN`：切换说明语言。
  - `X`：关闭弹窗。
- 键盘：`Esc` 关闭弹窗。

## 内容结构

只保留三段，不加入课堂技巧：

1. 设置游戏：Answers、Cups、Rounds、Speed。
2. 添加题目：手动输入、TXT 导入、图片文件夹导入、开始条件。
3. 怎么玩：看目标、观察洗牌、点击杯子、全部选对得分。

TXT 导入说明明确要求带序号，并加入轻提示：可以截图单词表后，让 AI 整理成带序号的文本。图片导入只写“pictures / 图片”，不写闪卡。

## 字体

现有内置 Nunito 字体不含中文字形。为避免中文说明显示方块字，`get_font` 在检测到 CJK 字符时优先尝试系统中文字体：

- macOS：PingFang / STHeiti
- Windows：Microsoft YaHei / SimHei / SimSun

英文 UI 继续优先使用项目内置字体。

## 验证

- `python3 -m compileall main.py game`
- headless 渲染 SettingsScreen、英文帮助弹窗、中文帮助弹窗
- 人工查看预览图，确认文字没有明显重叠，内容能完整显示
